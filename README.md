## install

###  Homebrew install
```bash
brew install --cask kweisoft/brew/uubrowser
```

OR:
```bash
brew tap kweisoft/brew
brew install --cask uubrowser
```

### Manual Installation

1. Download the latest DMG file from the [Releases](https://github.com/kweisoft/UUBrowser_mac/releases) page
2. Open the DMG file
3. Drag UUBrowser.app to your Applications folder
4. Launch UUBrowser from your Applications folder
